简易搭建VPS

发表于 2018-06-25 | 分类于 VPS

这篇笔记是以前就写好的，现在拿出来分享一下怎么搭建VPS。

准备工作

首先需要购买服务器，听朋友推荐vultr不错，我也就在vultr购买了。

首先到vultr官网，注册一个账号，然后新建server
第一步选择server location，不同的节点可以ping一下，一般是距离近速度较快。

第二步选择server type，本文是针对Debian等系统。

第三步选择server size，就看个人选择了。

之后步骤默认就好，点击deploy now。服务创建成功！当status显示为running，创建的服务就在正常运行了。

TCP Fast Open(可选)

这一步是为了增加速度。
在终端输入

然后在出现的界面中，在最下行添加如下

echo 3 > /proc/sys/net/ipv4/tcp_fastopen
然后按 Ctrl + X 退出，输入“Y”并回车确认退出。
然后

在最后添加

net.ipv4.tcp_fastopen = 3
然后退出

shadowsocks客户端
